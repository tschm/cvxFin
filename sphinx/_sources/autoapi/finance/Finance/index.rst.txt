.. py:function:: solveLP(c, A, bxl, bxu, bcl, bcu)

   Solves a linear program

   maximize    c'*x
   subject to  bcl <= A*x <= bcu
               bxl <=  x  <= bxu


.. py:function:: solveQPcon(c, A, Q, qc, bxl, bxu, bcl, bcu)

   Solves a convex program with a quadratic constraint

   maximize      c'*x
   subject to    bcl <= A*x <= bcu
                 bxl <=  x  <= bxu
               sqrt(x'*Q*x) <= qc


.. py:function:: solveQPobj(c, A, Q, bxl, bxu, bcl, bcu)

   Solves a convex program with a quadratic objective

   maximize    c'*x - (1/2) x'*Q*x
   subject to    bcl <= A*x <= bcu
                 bxl <=  x  <= bxu


.. py:function:: solveMarkowitzConstraint(c, A, Q, qc, v, x0, bxl, bxu, bcl, bcu)

   Solves a convex program with a quadratic constraint and linear abs penalty

   maximize     c'*x - v'*abs{x - x0}
   subject to    bcl <= A*x <= bcu
                 bxl <=  x  <= bxu
               sqrt(x'*Q*x) <= qc


.. py:function:: solveMarkowitzObjective(c, A, Q, v, x0, bxl, bxu, bcl, bcu)

   Solves a convex program with a quadratic constraint and linear abs penalty

   maximize     c'*x - (1/2) x'*Q*x - v'*abs{x - x0}
   subject to    bcl <= A*x <= bcu
                 bxl <=  x  <= bxu
